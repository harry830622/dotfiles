# dotfiles
My dotfiles

## Installation
1. Install git and clone this repo
1. Install alacritty
1. Install zsh
1. Install zim
1. Link zshrc and zimrc
1. Install nvm
1. Install vim-plug
1. Install vim
1. Link vimrc
1. Link coc settings
1. Install tmux
1. Install tpm
1. Link tmux.conf
